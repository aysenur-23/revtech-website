# Referans Logoları

Bu klasöre referans firmalarının logolarını yükleyin.

## Dosya Formatları
- **Önerilen**: PNG, SVG
- **Boyut**: 200x200px veya daha büyük
- **Arka Plan**: Şeffaf (transparent) tercih edilir

## Dosya Adlandırma
- `logo1.png` - 1. firma logosu
- `logo2.png` - 2. firma logosu  
- `logo3.png` - 3. firma logosu
- `logo4.png` - 4. firma logosu

## Kullanım
Logolar otomatik olarak hizmetlerimiz sayfasındaki "Referanslarımız" bölümünde görüntülenecektir.

## Fallback
Eğer logo dosyası bulunamazsa, placeholder logo gösterilecektir.
